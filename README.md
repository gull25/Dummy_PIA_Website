# PIA Website Clone

## Overview

This project is a front-end clone of the Pakistan International Airlines (PIA) website, developed using only HTML and CSS. The purpose of this project is to practice modern web design principles, responsive layouts, semantic HTML structure, and CSS styling techniques.

## Features

* Responsive navigation bar
* Hero section with background image
* Clean and modern user interface
* Responsive design for desktop, tablet, and mobile devices
* Semantic HTML5 structure
* Organized and maintainable CSS code
* Hover effects and interactive styling

## Technologies Used

* HTML5
* CSS3
